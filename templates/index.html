Deskripsi Kasus

Studi Kasus

Di dalam kehidupan mahasiswa, partisipasi dalam kegiatan sosial dan organisasi sangat penting untuk mengembangkan soft skills dan hard skills. Universitas ingin memahami tingkat keterlibatan mahasiswa dalam kegiatan-kegiatan tersebut untuk merancang program yang disesuaikan dengan kebutuhan masing-masing kelompok.

Tujuan

Mengelompokkan mahasiswa ke dalam tiga kategori—"Aktif," "Sedang," dan "Pasif"—berdasarkan tingkat keikutsertaan mereka dalam kegiatan sosial, organisasi kemahasiswaan, dan kepanitiaan. Dengan mengetahui kategori ini, universitas dapat merancang program pengembangan yang tepat sasaran untuk mendukung mahasiswa di setiap kelompok.

Data

Pada Proyek pengelompokan mahasiswa mengguanakan K-mean claustering ini memrlukan data dari mahasiswanya sedniri, dan data yang di ambil untuk mengkelompokanya adalah partisipasi sosial dan partisipasi organisasi, dengan ini dapat mengetahui apakah mahasiswa itu tergolong sebagai kategori —"Aktif," "Sedang," dan "Pasif"

Materi Implementasi


Implementasi K-Means Clustering Data Preprocessing: Mengubah data yang terkumpul ke dalam bentuk numerik, seperti skala untuk posisi yang dipegang (misalnya, anggota diberi nilai 1, koordinator nilai 2, dan ketuanilai 3). Normalisasi Data: Untukmenghindari bias darifiturdengan skala besar, data dinormalisasi agar setiap variabel memiliki skala yang seimbang.

Penerapan K-Means Clustering K-Means Clustering akan dijalankan dengan k = 3 untukmengelompokkan mahasiswa ke dalam tiga kategori: Aktif: Mahasiswa yang memiliki partisipasi tinggi di organisasi, sering mengikuti kegiatan sosial, dan biasanya memegang tanggung jawab lebih tinggi. Sedang: Mahasiswa dengan keterlibatan cukup aktif, tetapi tidak terlalu sering atau tidak selalu memegang peran penting. Pasif: Mahasiswa yang jarang terlibat organisasi dan kegiatan sosial. Umumnya tidak memegang tanggung jawab apapun.

Interpretasi dan Implementasi Hasil Ketiga kategori mahasiswa akan memiliki program yang berbeda, seperti: Kategori Aktif: Mahasiswa dalam kelompok ini dapat diarahkan ke program pelatihan pengembangan diri untuk memperkuat keahlian mereka. Kategori Sedang: Mahasiswa dengan partisipasi menengah dapat diberikan program pengembangan soft skill dasar, seperti komunikasi dan kerjasama sehingga termotivasi untuk lebih aktif lagi. Kategori Pasif: Kelompok ini dapat diikutkan dalam program motivasi atau pengenalan organisasi untuk mendorong partisipasi mereka secara bertahap.

Manfaat Pengelompokan Mahasiswa Pengelompokkan ini membantu universitas dalam: Menargetkan program pengembangan yang sesuai untuk setiap kategori mahasiswa. Menyusun strategi peningkatan partisipasi mahasiswa di lingkungan kampus. Mengembangkan soft skills dan hard skills mahasiswa. Mempersiapkan mahasiswa yang tangguh dan siap untuk menghadapi dunia kerja.

Pengimplementasian Python adalah pilihan ideal untuk proyek ini karena proyek pustakanya yang luas. Langkah-langkahnya meliputi preprocessing data dengan pandas dan NumPy, normalisasi menggunakan scikit-learn, serta penerapan algoritma K-Means di scikit-learn. Hasil clustering dianalisis secara visual menggunakan matplotlib dan seaborn. Untuk penyimpanan data, MySQL atau SQLite dapat dikelola dengan SQLAlchemy. Jika diperlukan skalabilitas, platform seperti Google Colab atau AWS SageMaker bisa digunakan. Hasil akhir dapat disajikan melalui dashboard interaktif dengan Dash atau Streamlit untuk mempermudah eksplorasi wawasan.
